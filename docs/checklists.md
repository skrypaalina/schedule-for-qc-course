# Чеклісти для тестування
## Модуль: CRUD викладачів та кафедр (Варіант 5)

### Функціональність
- [ ] Створення нової кафедри з унікальною назвою.
- [ ] Додавання викладача з прив'язкою до існуючої кафедри.
- [ ] Редагування ПІБ викладача та перевірка збереження змін.
- [ ] Відображення списку викладачів після фільтрації за кафедрою.
- [ ] Видалення кафедри, яка не має прив'язаних викладачів.
- [ ] Пошук викладача за прізвищем (частковий збіг).

### Негативні сценарії
- [ ] Спроба зберегти викладача з порожнім обов'язковим полем ПІБ.
- [ ] Введення цифр та спецсимволів у поле імені викладача.
- [ ] Введення тексту довжиною понад 50 символів у поле ПІБ.
- [ ] Спроба створити кафедру-дублікат.
- [ ] Видалення викладача, який призначений на активну пару в розкладі.
- [ ] Спроба зберегти форму викладача без вибору кафедри.

### UI/UX
- [ ] Наявність модального вікна підтвердження при видаленні.
- [ ] Відображення повідомлень про помилки валідації червоним кольором.
- [ ] Адаптивне відображення таблиці викладачів на мобільних пристроях.
